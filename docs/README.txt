BÁO CÁO QUÁ TRÌNH THỰC HIỆN

Họ và tên: Đặng Bình Đại

MSSV: 231CTT038

Môn học: Thiết kế Website

1. Công cụ AI đã sử dụng

* ChatGPT hỗ trợ giải thích yêu cầu đề bài.
* ChatGPT hỗ trợ xây dựng giao diện HTML, CSS và JavaScript.
* ChatGPT hỗ trợ xử lý lỗi giao diện và hướng dẫn triển khai GitHub Pages.

2. Các prompt chính đã sử dụng

* Thiết kế giao diện Dashboard học tập bằng HTML, CSS và JavaScript.
* Tạo Sidebar sử dụng Flexbox.
* Tạo Course Grid sử dụng CSS Grid.
* Viết JavaScript hiển thị môn học đang chọn khi nhấn nút Xem.
* Viết JavaScript đổi màu menu Sidebar khi được chọn.
* Viết JavaScript tự động thay đổi số cột Grid dựa trên MSSV.
* Hướng dẫn tạo GitHub Repository và GitHub Pages.

3. Những phần tự thực hiện

* Tạo cấu trúc thư mục dự án theo yêu cầu đề bài.
* Tạo Repository trên GitHub.
* Tạo và chỉnh sửa các file index.html, style.css và script.js.
* Chuẩn bị hình ảnh minh họa cho các môn học.
* Chỉnh sửa thông tin cá nhân (họ tên, MSSV).
* Kiểm tra giao diện và sửa các lỗi hiển thị.
* Đưa mã nguồn lên GitHub và triển khai GitHub Pages.

4. Những phần tham khảo AI

* Tham khảo cách bố trí giao diện Dashboard.
* Tham khảo mã CSS cho Flexbox và Grid.
* Tham khảo JavaScript xử lý sự kiện nút bấm.
* Tham khảo cách thay đổi số cột Grid theo MSSV.
* Tham khảo hướng dẫn sử dụng GitHub Pages.

5. Khó khăn gặp phải

* Chưa quen với cách tổ chức dự án Website.
* Gặp lỗi đường dẫn hình ảnh và file CSS.
* Khó khăn trong việc căn chỉnh giao diện giống mẫu.
* Chưa nắm rõ cách triển khai GitHub Pages.
* Cần thời gian tìm hiểu JavaScript để xử lý các chức năng tương tác.

6. Kết quả đạt được

* Hoàn thành giao diện Dashboard học tập theo yêu cầu.
* Sử dụng Flexbox cho Header, Sidebar và khu vực thống kê.
* Sử dụng CSS Grid để hiển thị danh sách môn học.
* Thực hiện thành công chức năng hiển thị môn học được chọn.
* Thực hiện thành công chức năng đổi màu menu Sidebar.
* Tự động thay đổi bố cục Grid theo MSSV bằng JavaScript.
* Đưa Website lên GitHub Pages và truy cập thành công bằng đường dẫn username.github.io.
